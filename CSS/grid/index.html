CSS Grid 网格布局参考

父类属性

grid-template-columns + column-gap

grid-template-columns该属性用于定义列数和宽度。您既可以单独设置每列的宽度，也可以使用 repeat() 函数为所有列设置统一的宽度

column-gap 此属性用于在网格内的列之间放置间隙

grid-template-rows + row-gap

grid-template-rows该属性用于定义行数和宽度。您既可以单独设置每行的高度，也可以使用 repeat() 函数为所有行设置统一的高度

row-gap 此属性用于在网格内的行之间放置间隙

align-items + justify-items = place-items

justify-items属性设置单元格内容的水平位置（左中右）start | end | center | stretch

align-items属性设置单元格内容的垂直位置（上中下）start | end | center | stretch

place-items属性是俩属性合并简写
